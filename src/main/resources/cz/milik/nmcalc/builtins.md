Builtins
========

Builtin Functions
-----------------

### Mathematical Functions

  - `mod(a, b)` - modulo
  - `lcm(a, b)` - least common multiple

  - `rand()` - random

  - `pow(b, e)` - power
  - `sqrt(value)` - square root
  - `ln(value)` - natural logarithm
  
  - `abs(value)` - absolute
  - `ceil(value)` - ceiling
  - `floor(value)` - floor
  - `sin(value)` - sine
  - `asin(value)` - arcsine
  - `cos(value)` - cosine
  - `acos(value)` - arcosine
  - `tan(value)` - tangent
  - `atan(value)` - arctangent

### Other Functions

  - `len(collection)` - length/size of a collection or string
  
  - `some(value)`

  - `quote(value)`
  - `list(...)` - create list
  - `cons(head, tail)` - join a head and a tail
  - `head(list)` - return the head of a list
  - `tail(list)` - return the tail of a list

  - `dict(list)` - construct a dictionary
  
  - `apply`
  - `unapply`
  - `eval`

  - `hex(value)`
  - `oct(value)`
  - `bin(value)`

  - `env()`

  - `help([value])`
